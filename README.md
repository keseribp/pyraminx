# triangular_pyramid_puzzle

## Build: 
`$ nvcc -std=c++11 main.cpp -o rubikPuzzle -lglfw3 -lGLU -lGL -lGLEW -lX11 -lXxf86vm -lXinerama -lXcursor -lrt -lm -lXrandr -lpthread -lXi -ldl`
